## Purpose

Write feature announcements that **drive adoption, not just awareness** — tailored for different channels (in-app, email, blog, social) with user-centric messaging.

---

## Application

### Multi-Channel Announcement Template

```markdown
# Feature Announcement: [Feature Name]
**Launch Date:** [Date]
**Target Users:** [Who benefits most]

---

## Core Messaging

### Headline (8 words max)
[Action-oriented: what users can now DO]

### One-liner
[Benefit-first description in one sentence]

### Detailed Description (2-3 sentences)
[What it does + why you'll care + how to try it]

---

## Channel-Specific Copy

### 📱 In-App Notification
```
**[Headline]**
[One-liner]
[CTA Button: "Try it now →"]
```

### 📧 Email
```
Subject: [Action verb] + [benefit] — now available
Preview text: [First line hook]

Hi [Name],

[One paragraph: What changed and why you'll care]

[One paragraph: How to use it — 3 simple steps or a GIF]

[CTA Button: "Try [Feature Name]"]

— [Team Name]
```

### 📝 Blog Post
```
# [Headline — benefit-focused title]

[Opening paragraph: The problem this solves]
[Middle: How it works — with screenshots]
[Context: Why we built this — user feedback/data]
[Close: What's next — roadmap tease]
[CTA: Try it now]
```

### 🐦 Social
```
[Short hook — benefit or stat]
[Feature name + one-liner]
[Link]
[Optional: screenshot or short video]
```
```

### Writing Rules

| Do | Don't |
|----|-------|
| "You can now [action]" | "We built [feature]" |
| "Save 2 hours per week" | "Improved efficiency" |
| Show a screenshot or GIF | Just describe it |
| Include a CTA | Hope they find it |
| Lead with benefit | Lead with technology |

---

## Quality Rubric

| Dimension | 0 | 1 | 2 |
|-----------|---|---|---|
| **Messaging** | Feature-focused | Benefit-mentioned | Benefit-first with proof |
| **Channels** | One channel only | 2-3 channels | All relevant channels tailored |
| **CTA** | No CTA | Generic ("Learn more") | Specific action ("Try X now") |
| **Visuals** | Text only | Screenshot | GIF/video showing feature |
