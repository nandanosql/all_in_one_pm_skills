## Purpose

Prepare for PM interviews across **all question types** — product sense, estimation, analytics, behavioral, strategy, and execution — with structured frameworks, practice questions, and feedback.

---

## Application

### Question 1: Interview Type

**Agent asks:** "What type of PM interview are you preparing for?"

1. **Product sense** — "How would you improve X?" / "Design a product for Y"
2. **Estimation / Analytics** — "How many X are sold per year?"
3. **Behavioral** — "Tell me about a time when..."
4. **Strategy** — "What should company X do about Y?"
5. **Execution** — "How would you ship X?"
6. **Full mock** — Practice all types

### Question 2: Level

**Agent asks:** "What level are you targeting?"

1. APM / Entry-level
2. Mid-level PM
3. Senior PM
4. Director+

### Practice Flow

For each question type, agent:
1. Asks a practice question
2. User responds
3. Agent provides structured feedback:
   - What was strong
   - What to improve
   - Model answer framework

### Frameworks Quick Reference

```markdown
**Product Sense:** User → Problem → Solutions → Prioritize → Metrics
**Estimation:** Clarify → Decompose → Calculate → Sanity Check
**Behavioral:** STAR (Situation → Task → Action → Result)
**Strategy:** Market → Competition → Capabilities → Recommendation
**Execution:** Goal → Requirements → Timeline → Risks → Metrics
```
