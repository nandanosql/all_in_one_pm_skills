## Purpose

Design and facilitate effective product workshops — providing a **proven facilitation protocol** that can be adapted to any topic (strategy, discovery, prioritization, retrospective, etc.).

---

## Application

### Workshop Planning Template

```markdown
# Workshop Plan: [Topic]

## Setup
- **Goal:** [What will be decided/produced by end]
- **Attendees:** [Who + why they're needed]
- **Duration:** [Total time]
- **Format:** [In-person / Remote / Hybrid]
- **Pre-work:** [What attendees should prepare]

## Agenda
| Time | Activity | Format | Facilitation Notes |
|------|----------|--------|-------------------|
| 0:00 | Welcome + ground rules | Plenary | Set expectations |
| 0:05 | Context setting | Presentation | 5 min max |
| 0:10 | [Activity 1] | [Solo/pairs/group] | [Instructions] |
| 0:25 | Share out | Each person 2 min | [Capture on board] |
| 0:40 | [Activity 2] | [Format] | [Instructions] |
| 0:55 | Dot voting / prioritization | Individual → group | [Voting rules] |
| 1:05 | Action items + next steps | Plenary | [Who does what by when] |
| 1:15 | Close | — | Thank participants |

## Ground Rules
1. One conversation at a time
2. Ideas before judgment
3. Timeboxes are respected
4. Everyone contributes
5. Decisions are documented

## Materials
- [ ] Whiteboard / Miro board
- [ ] Timer
- [ ] Sticky notes (physical or digital)
- [ ] Voting dots
- [ ] Pre-prepared templates

## Facilitation Moves
| Situation | Move |
|-----------|------|
| One person dominates | "Thanks [name], let's hear from others" |
| Silence | "Take 2 min to write individually, then share" |
| Off-topic | "Great point — parking lot it and return after" |
| Conflict | "Let's capture both perspectives and test with data" |
| Running long | "Let's timebox this to 5 more min and decide" |
```

---

## Quality Rubric

| Dimension | 0 | 1 | 2 |
|-----------|---|---|---|
| **Goal Clarity** | No clear outcome | Goal stated | Goal + success criteria |
| **Structure** | Freeform discussion | Timed agenda | Activities + formats + facilitation notes |
| **Participation** | PM monologue | Some input | Structured for equal contribution |
| **Outputs** | "Good discussion" | Ideas captured | Action items + owners + due dates |
