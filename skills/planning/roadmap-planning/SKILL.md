## Purpose

Build a **strategic product roadmap** from input gathering through stakeholder communication. Orchestrates prioritization, capacity planning, epic definition, and dependency mapping into a coherent planning process.

## Common Pitfalls

### Pitfall 1: Roadmap as Feature List

**Fix:** Every roadmap item should tie to a strategic objective. If it doesn't, it either doesn't belong or you're missing a strategic connection.

### Pitfall 2: Over-Promising

**Fix:** Always capacity-plan before committing. Build in 10-15% buffer for unknowns.
